## Clauderfall Product Brief Reference

Use this reference when you need the product-level framing for Design.

Core product:

* Clauderfall helps a single senior engineer turn rough software ideas into strong discovery briefs and design artifacts.
* The current product boundary is Discovery plus Design.
* The product's purpose is to improve problem framing and design quality before execution begins.

Core design role:

* Design is the stage where solution structure becomes concrete.
* Design should make the solution concrete enough for downstream implementation planning without forcing later stages to invent core design decisions.
* Design artifacts should remain readable first, while carrying some structured side where it materially improves downstream planning and refinement.

Approval points:

* Discovery-to-Design handoff approval
* design-unit build-readiness approval
