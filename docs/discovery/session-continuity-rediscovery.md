# Session Continuity Rediscovery

## Overview

This is a stub Discovery for session continuity as a separate problem from the broader persistence substrate.

It exists to keep startup cost, handoff shape, continuation safety, and startup skill behavior out of the persistence-substrate Discovery until the substrate question is framed first.

## Intended Scope

This Discovery should later cover:
- startup orientation cost
- active-thread handoff shape
- continuation-safe carry-forward state
- session startup and handoff skill behavior
- how to keep recent-session context bounded without losing useful continuity

## Current Readiness

Not ready for Design.

This artifact is intentionally a stub pending the persistence-substrate Discovery.
